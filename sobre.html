Home Sobre

Este projeto foi desenvolvido como parte de um estudo sobre desenvolvimento web. Ele tem como objetivo fornecer uma calculadora de gasto calórico baseada na equação de Harris-Benedict.

O código HTML, CSS e JavaScript foi criado para demonstrar como criar um formulário de entrada de dados, realizar cálculos com base nesses dados e exibir os resultados na página.

É importante ressaltar que este projeto é apenas educacional e não deve ser utilizado para orientação médica ou nutricional. Consulte sempre um profissional de saúde qualificado para orientações personalizadas.
